# 🐳 Guia Docker

## 🔧 Serviços

- **API Modelo (`api_model`)**  
  - Porta: 8080  
  - Imagem baseada em `python:3.8-slim`

- **Frontend (`front_local`)**  
  - Porta: 80  
  - Servidor Nginx serve os arquivos da pasta `frontend`

## 🔗 Comandos úteis

```bash
docker-compose up --build
docker-compose down
```
